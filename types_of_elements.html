Types of Element - 1) Block Element: -> These elements always start on a new line. -> Occupies the entire available width. -> Top and bottom Margins exist. Examples: - p - h1-h6 - ul - li - blockquote - dd - dl - dt - tfoot - address - pre 2) Inline Element: -> These elements continue on the current line. -> These take up as much width as required. -> Top and bottom Marginsdo not exist. Examples: - span - b - abbr - sup - i - q
